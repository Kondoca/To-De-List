# To-De-List

https://kondoca.github.io/To-De-List/

Um aplicativo de lista de tarefas simples e elegante criado com HTML, CSS e JavaScript puro.

## Descrição

Este projeto permite que você adicione, edite, marque como concluída e exclua tarefas. Ele usa `localStorage` para manter a lista mesmo após fechar ou atualizar o navegador.

## Recursos

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Editar tarefas diretamente na lista
- Filtrar tarefas por:
  - Todas
  - Pendentes
  - Feitas
- Limpar todas as tarefas concluídas
- Marcar todas as tarefas como concluídas
- Contador de tarefas pendentes
- Suporte responsivo e acessível

## Estrutura de arquivos

- `index.html` - estrutura da interface
- `styles.css` - estilo visual e responsividade
- `app.js` - lógica de tarefas, renderização e persistência

## Como usar

1. Abra `index.html` no seu navegador.
2. Digite uma tarefa no campo de entrada.
3. Clique em `➕` ou pressione `Enter` para adicionar.
4. Use os botões de filtro para ver tarefas pendentes, concluídas ou todas.
5. Clique no ícone de lixeira para excluir uma tarefa.
6. Clique em uma tarefa para editá-la e pressione `Enter` ou clique fora para salvar.

## Tecnologias

- HTML
- CSS
- JavaScript

## Observações

O projeto funciona localmente sem necessidade de servidor. Basta abrir o arquivo `index.html` em qualquer navegador moderno.
